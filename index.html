TCUWebDataCache

A simple web cache objective-c iOS library designed mainly for image caching.

Example usage is alike:

TCUWebDataCache *webDataCache = [TCUWebDataCache sharedDataCache]; BOOL isCached = [webDataCache isDataFromURLCached:@"http://example.com/image.png"]; NSData *imageData = [webDataCache getDataFromURL:inboundImageURLString cached:YES]; UIImage *image = [UIImage imageWithData:imageData];
